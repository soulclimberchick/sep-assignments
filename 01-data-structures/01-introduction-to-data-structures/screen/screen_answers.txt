Q: How does your data structure allow developers to access and manipulate the data?

A: Developers are able to use attribute accessors that are instance variables
so that they can keep the pixel and screen classes separate and clean and reusable.
The attribute attr_accessor are set so that the developer can enter standard
numeric values for both the colors and the pixel coordinates.

Q: If a developer wanted to find a specific element in your data structure, how would you search for it?

A: The developer could search by pixel coordinates to determine the
color values.
